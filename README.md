# :brain: knowledgebase :brain:

## Purpose

This repository is a software development knowledge building collaboration space.

## Contribution Guidelines

To contribute to this repository as a member of the friendcorp org, please make your changes in a new branch of this repository and create a pull request when completed. Non-members will need to fork the repository prior to the above steps.

More information about best github collaboration practices available in [this video](https://www.youtube.com/watch?v=MnUd31TvBoU&ab_channel=TheNetNinja).

## General

- Video - [Web Development In 2022 - A Practical Guide](https://youtu.be/EqzUcMzfV1w)
  - This is a very high-level overview of common web development technologies and practices in 2022.
  - Also contains helpful insights for identifying an individuals web development goals and ideal path.
- Curriculum - [freeCodeCamp.org](https://www.freecodecamp.org/learn/)
  - This is a well regarded organization that provides a large number of specialized courses and certifications.
  - This resource also has coding interview prep tools.
- Website / Organization - [MDN Web Docs](https://developer.mozilla.org/en-US/)
  - This website is an excellent source of documentation relating to web development.
  - If looking for documentation on a particular area of web development, this is a trustworthy resource.
- Website - [Stack Overflow](https://stackoverflow.com/)
  - If you find yourself stuck, search this website and see if someone has provided a solution to a similar issue here in the past.
